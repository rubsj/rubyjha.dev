This is a personal portfolio site. I've kept data collection to an absolute minimum.

## Analytics

This site uses **Cloudflare Web Analytics** — a privacy-respecting analytics tool that:

- Does not use cookies
- Does not collect personal data
- Does not track visitors across sites
- Does not fingerprint browsers
- Is fully compliant with GDPR, CCPA, and similar privacy regulations

I only see aggregate traffic data (page views, top pages, general geography). No personal information is collected or stored.

## Comments (Future)

A future version of this site will include blog comments powered by **Giscus**, which uses GitHub Discussions. If you choose to leave a comment, GitHub authentication is required. Giscus is subject to [GitHub's Privacy Policy](https://docs.github.com/en/site-policy/privacy-policies/github-general-privacy-statement).

## External Links

This site links to third-party services (LinkedIn, GitHub, etc.). Those sites have their own privacy policies and I have no control over their data practices.
